# Ukrainian Payments Analytics
Аналітика платіжної поведінки українців у воєнній економіці на основі SQL. Статус: Робота триває.
# Опис проєкту
Як повномасштабна війна змінила платіжний ринок в Україні?
Аналіз структуриних зрушень на ринку безготівкових платежів в 2020-2025 роках, на основі офіційних даних Національного Банку України. Проект досліджує наступні явища: прискорення процесу цифровізації платежів, зміну балансу на банківському ринку та регіональні скорочення платіжної інфраструктури за областями.

# Мета
Повномасштабне вторгнення 2022 року стало значним шоком для платіжної інфраструктури, банків і економіки в цілому. Одні показники пришвидшились, інші безповоротно впали, конкуретний баланс між банками змінився кардинальним чином.
Метою проекту є на основі публічних даних НБУ проаналізувати основні зміни в платіжній поведінці українів протягом досліджуваного періоду, конкурентні зміни на банківському ринку, що саме змінилося, як і на скільки стійкими є ці зміни.
# Основні питання
- Чи прискорила війна відмову від готівки?
- Хто на банківському ринку виграв конкурентну гонку в кризовий період?
- Які структурні зміни відбулися в платіжній інфраструктурі регіонів?
